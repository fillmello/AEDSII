# AEDSII

Repositorio de estudos da disciplina de Algoritmos e Estruturas de Dados II (AEDSII).

Aqui ficam organizadas solucoes, testes e experimentos com algoritmos vistos em aula, principalmente em Java.

## Estrutura

- `src/` - codigos-fonte dos exercicios e algoritmos
- `src/BubbleSort.java` - implementacao do Bubble Sort
- `src/InsertionSort.java` - implementacao do Insertion Sort

## Como executar

Se estiver trabalhando com os arquivos Java dentro de `src/`, voce pode compilar e executar assim:

```bash
javac src/InsertionSort.java
java -cp src InsertionSort
```

Para o Bubble Sort:

```bash
javac src/BubbleSort.java
java -cp src BubbleSort
```

## Objetivo

O repositorio serve para praticar:

- ordenacao de vetores
- analise de desempenho
- estrutura basica de programas em Java
- testes com entradas aleatorias

## Observacoes

- Os arquivos podem ser adaptados conforme os exercicios da disciplina.
- Se necessario, novos algoritmos podem ser adicionados na pasta `src/`.
